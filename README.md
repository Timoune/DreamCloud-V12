# DreamCloud V12

Cognitive Memory Subsystem for the Mini Von Architecture.

## Features
- Persistent long-term memory with SQLite + FAISS
- Semantic retrieval with graph expansion
- DreamCycle concept synthesis
- Reinforcement & decay dynamics
- NLI contradiction detection
- Typed memories

## Setup
1. `pip install -r requirements.txt`
2. Start llama.cpp server on port 8080
3. `python main.py`
